# React + TypeScript + Vite Boilerplate

This template provides a minimal setup to get React working in Vite with HMR and some ESLint/Prettier rules.

Licensed under [MIT](LICENSE) — use freely for any purpose.

## Getting Started

This project has dependencies that are required to function correctly:

```bash
pnpm install
```

### Update project meta

When using this as a starter, update the `name` field in `package.json` to match your project.

### Recommended IDE extensions (Cursor / VS Code)

This project recommends extensions for formatting, linting, and spell checking. When you open the workspace, Cursor/VS Code will prompt you to install them.

| Extension                            | Purpose                                   |
| ------------------------------------ | ----------------------------------------- |
| Prettier                             | Code formatting (runs on save)            |
| ESLint                               | Linting (fixes on save)                   |
| Code Spell Checker                   | Spell checking with British English       |
| British English - Code Spell Checker | British dictionary (colour, centre, etc.) |

These are optional — the project runs fine without them, but you'll miss format-on-save, lint-on-save, and spell checking.

### Running the local dev server

```bash
pnpm run dev
```

### Building and previewing

```bash
pnpm run build
pnpm run preview
```

### ESLint/Prettier

Both tools auto-run when saving a document. To run manually:

```bash
pnpm run lint
pnpm run lint:fix
pnpm run format
```

### Testing

```bash
pnpm run test
pnpm run test:run
```
